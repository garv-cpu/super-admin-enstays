# Enstays Super Admin

React JS + Vite + Tailwind CSS web application starter for the Enstays Super Admin panel.

## Use inside Enstays

Place this folder here:

```txt
Enstays/
  apps/
    super-admin/
```

## Run

```bash
cd apps/super-admin
npm install
npm run dev
```

## Current scope

- Web app only for Super Admin.
- No backend code included.
- Placeholder screens prepared for:
  - Dashboard
  - Hotel Groups
  - Hotel Brands
  - Properties
  - PMS Providers
  - Platform Users
  - Roles & Permissions
  - Audit Logs

## Environment

Copy `.env.example` to `.env` when backend is ready.

```bash
cp .env.example .env
```
